# React Hello World

一个简单的 React Hello World 应用程序，使用 Vite 构建。

## 环境搭建

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:5173` 运行

### 3. 构建生产版本

```bash
npm run build
```

### 4. 预览生产构建

```bash
npm run preview
```

## 项目结构

```
react-hello-world/
├── src/
│   ├── App.jsx       # 主 React 组件
│   ├── main.jsx      # 入口文件
│   └── index.css     # 全局样式
├── index.html        # HTML 模板
├── package.json      # 依赖和脚本
├── vite.config.js    # Vite 配置
└── .gitignore        # Git 忽略规则
```

## 技术栈
